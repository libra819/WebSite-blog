
# 個人技術部落格專案

## 專案簡介
此專案為一個用於整理與分享技術學習歷程的個人部落格系統，主要記錄在開發過程中遇到的問題、解決方式、程式碼範例以及相關技術筆記。內容涵蓋前後端開發經驗、實務應用案例與實作心得，期望能將零散的知識整理成可查詢、可複用的技術紀錄。

除了技術文章之外，部分內容亦會紀錄開發學習過程中的生活觀察與思考，讓整體內容不僅限於程式技術，也保留成長與探索的軌跡。

本專案亦希望提供一個可參考的部落格架構範例，讓有需要建立個人技術網站的人可以快速理解整體系統架構並進行延伸開發。

---

## 系統架構

### 前端
- **Angular 20**
- 用於建立現代化單頁應用（SPA）介面
- 負責文章展示、分類瀏覽、搜尋與閱讀體驗
- 使用元件化設計提升可維護性與擴充性

### 後端
- **Node.js + Express**
- 提供 RESTful API
- 負責文章資料管理、內容讀取與系統服務
- 作為前端與資料層之間的橋接

## 專案內容

部落格主要包含以下內容：

- 技術文章整理
- 程式問題解決紀錄（Troubleshooting）
- 開發經驗與實作流程

- 開發工具與技術研究筆記
- 偶爾更新的生活紀錄
